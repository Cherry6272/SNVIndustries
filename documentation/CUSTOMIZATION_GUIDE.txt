🛠️ TOOLS YOU'LL NEED:

1. Text Editor (Choose one):
   - Notepad++ (Windows) - Free
   - Sublime Text - Free
   - VS Code - Free
   - Even regular Notepad works!

2. Image Editor (Optional):
   - Canva.com - Free, online
   - GIMP - Free desktop app
   - Photoshop - Paid

3. FTP Client (for uploading):
   - FileZilla - Free
   - Or use cPanel File Manager


⚠️ IMPORTANT TIPS:

✓ Always save a backup before editing
✓ Use Find & Replace for bulk changes
✓ Test locally before uploading
✓ Keep file names exactly the same
✓ Don't change file structure
✓ Clear browser cache after changes
